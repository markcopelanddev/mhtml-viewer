# Changelog

## 1.2.0

- Added sort controls for Name, Date modified, Type, Size, Date created, Authors, Categories, Tags, and Title.
- Added Ascending/Descending ordering.
- Preserved the currently open file when changing sort mode and synchronized the bottom slider to the new position.
- Added lightweight metadata sorting for MHTML/PDF title-style fields when available.
- Preserved `1.1.0` under `legacy/1.1.0`.

## 1.1.0

- Added a `Select File` button that loads the selected file's folder and starts at that file.
- Added a bottom slider for jumping through all supported files in the selected folder.
- Kept Previous/Next navigation, keyboard navigation, natural sorting, and 1%/5% zoom controls.

## 1.0.0

- Initial Windows desktop app.
- Select a folder of `.mhtml`, `.mht`, or `.pdf` files.
- Move backward and forward through naturally sorted files.
- Zoom with 1% and 5% controls.
